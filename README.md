# ChatWork Threader

ChatWorkのメッセージをスレッド形式でツリー表示するChrome拡張機能です。

## 🎯 機能

- 返信関係のあるメッセージをツリー構造で表示
- メッセージの種類を色分けして表示
  - 🟢 **返信元**: 返信がついているメッセージ
  - 🔵 **返信**: 他のメッセージへの返信
  - 🟡 **返信元+返信**: 両方の役割を持つメッセージ
- スレッドをクリックで該当メッセージへスムーズスクロール
- リアルタイムでスレッド情報を更新

## 📦 インストール方法

### 開発者モードでのインストール

1. このリポジトリをクローン
   ```bash
   git clone https://github.com/blocquality/ChatWorkThreader.git
   ```

2. Chromeで `chrome://extensions/` を開く

3. 右上の「デベロッパーモード」をONにする

4. 「パッケージ化されていない拡張機能を読み込む」をクリック

5. クローンしたフォルダを選択

## 🚀 使い方

1. ChatWork（https://www.chatwork.com/）にアクセス
2. チャットルームを開く
3. 右下に表示される 🌳 ボタンをクリック
4. 右側にスレッドパネルが開きます
5. スレッド内のメッセージをクリックすると、該当メッセージにスクロールします

## 📝 メッセージの種類

| 種類 | 説明 |
|------|------|
| 返信元でも返信先でもない | スレッドに含まれない独立したメッセージ |
| 返信元 | 誰かから返信がついているメッセージ |
| 返信先 | 他のメッセージに対する返信 |
| 返信元+返信先 | 返信でありかつ、さらに返信がついているメッセージ |
